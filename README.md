# PromptConduit Claude Code Plugins

Official [Claude Code](https://claude.com/claude-code) plugin marketplace for [PromptConduit](https://promptconduit.dev).

## Install

In Claude Code, add the marketplace once:

```
/plugin marketplace add promptconduit/claude-plugins
```

Then install the plugin:

```
/plugin install promptconduit@promptconduit
```

## Plugins

| Plugin | Description |
|--------|-------------|
| `promptconduit` | Skills and helpers for capturing AI assistant events with PromptConduit |

### `promptconduit`

Includes:

- **`/promptconduit:getting-started`** — Walks you through installing the CLI, authenticating, and wiring up your AI assistants (Claude Code, Cursor, Gemini CLI).

More skills coming soon. Contributions welcome.

## What is PromptConduit?

PromptConduit captures raw events from AI coding assistants and forwards them to the PromptConduit platform for observability, analytics, and replay. The CLI is open source and the platform is hosted at [promptconduit.dev](https://promptconduit.dev).

- **CLI:** [github.com/promptconduit/cli](https://github.com/promptconduit/cli)
- **macOS app:** [github.com/promptconduit/app](https://github.com/promptconduit/app)
- **Homebrew tap:** [github.com/promptconduit/homebrew-tap](https://github.com/promptconduit/homebrew-tap)
